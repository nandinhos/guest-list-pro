# Skill: Laravel Analyzer

## Objetivo
Analisar código Laravel com contexto pré-mapeado

## Input
- Lista de arquivos

## Ações

1. Identificar responsabilidades
2. Detectar:
   - Violação de SRP
   - Acoplamento
   - Uso indevido de Facades
   - Regras no lugar errado

## Output

Salvar em:
.refactor/analysis/{file}.json
