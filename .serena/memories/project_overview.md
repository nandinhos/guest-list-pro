# Visão Geral do Projeto - Guest List Pro

## Propósito
O Guest List Pro é um sistema de gerenciamento de listas de convidados (guest lists).

## Tech Stack
- **Framework**: Laravel 12
- **Painel Administrativo**: Filament v4
- **Frontend Stack**: TALL Stack (Tailwind CSS, Alpine.js, Laravel, Livewire)
- **Infraestrutura**: Docker usando Laravel Sail
- **Banco de Dados**: MySQL 8.0
- **Linguagens**: PHP 8.4, JavaScript
