# Estilos e Convenções de Código

## Diretrizes Gerais
- Seguir os princípios **SOLID** e **Clean Code**.
- Nomenclatura de classes e métodos em **Inglês** (padrão Laravel).
- Comentários no código obrigatoriamente em **Português**.
- Uso de Type Hints sempre que possível.

## Convenções Laravel/Filament
- Seguir os padrões oficiais do Laravel 12.
- Utilizar Resources do Filament para o painel administrativo.
- Manter a lógica de negócio preferencialmente em Services ou Actions.
