# Checklist de Finalização de Tarefa

Ao concluir uma tarefa neste projeto, siga estes passos:
1. Executar testes automatizados: `./vendor/bin/sail artisan test`.
2. Verificar linting do código: `./vendor/bin/sail ./vendor/bin/pint`. (opcional, mas recomendado)
3. Realizar o commit com mensagem clara em **Português**.
4. Atualizar a documentação ou walkthrough se houver mudanças significativas.
