# DealDeck Design Kit v1.0.0

Analytics SaaS design kit with electric blue accent and clean data-forward aesthetics.

## Components

| Component | File | Description |
|-----------|------|-------------|
| `<x-dash-blue.button>` | button.blade.php | Multi-variant action button |
| `<x-dash-blue.card>` | card.blade.php | Content container with soft shadow |
| `<x-dash-blue.badge>` | badge.blade.php | Status and label badges |
| `<x-dash-blue.input>` | input.blade.php | Form input with label and error |
| `<x-dash-blue.alert>` | alert.blade.php | Feedback alerts (info/success/error/warning) |
| `<x-dash-blue.stat-card>` | stat-card.blade.php | KPI card with trend indicator |
| `<x-dash-blue.avatar>` | avatar.blade.php | User avatar with initials fallback |
| `<x-dash-blue.table-row>` | table-row.blade.php | Table row with hover state |
| `<x-dash-blue.sidebar-item>` | sidebar-item.blade.php | Navigation sidebar item |
| `<x-dash-blue.progress>` | progress.blade.php | Progress bar with label |

## Design Tokens

- **Primary**: `#4D5EF6` (Electric Blue)
- **Background**: `#E9ECF1`
- **Font**: Inter
- **Radius**: `rounded-2xl` / `rounded-3xl`
- **Shadow**: `0 4px 20px rgba(0,0,0,0.05)`
